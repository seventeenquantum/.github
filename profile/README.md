# 17 Quantum Education

> Preparing Africans for the next computing revolution.

We are Africa's quantum computing education platform — building the talent, tools, and community to ensure the continent does not just participate in the quantum era, but leads within it.

Through hands-on simulations, coding exercises, and access to real quantum hardware, we equip learners with practical skills to design and implement quantum algorithms that solve real problems in healthcare, energy, and agriculture across Africa.

---

## What We Do

We offer free and low-cost access to structured quantum computing courses designed specifically for African learners — from complete beginners to advanced practitioners. Our curriculum spans quantum mechanics fundamentals, quantum programming languages, algorithm design, and applied quantum problem-solving.

---

## This Organisation

This GitHub organisation hosts the open course repositories for all 17 Quantum Education programmes.

Each public repository corresponds to a single course and follows a consistent structure:

| Branch | Purpose |
|---|---|
| `main` | Starter code and exercises — fork this to begin |
| `solution` | Completed reference implementation — consult after submission |

**Learners do not need a GitHub account with org membership.** All course repos are public. Fork the repo, complete the exercises, and submit your work as a pull request back to `main`.

---

## How to Get Started

```bash
# 1. Fork the course repo on GitHub

# 2. Clone your fork
git clone https://github.com/<your-username>/<course-repo>.git
cd <course-repo>

# 3. Create a working branch
git checkout -b submission/<your-name>

# 4. Complete the exercises, then push
git add .
git commit -m "feat: complete exercise 01"
git push origin submission/<your-name>

# 5. Open a pull request to main on the upstream course repo
```

Each repo contains a `CONTRIBUTING.md` with course-specific setup instructions and submission guidelines.
